# 社区行为准则

## 我们的承诺

HotKey 社区致力于为所有参与者提供友善、安全、尊重且不受骚扰的协作环境，不因年龄、身体状况、民族、性别认同与表达、经验水平、教育背景、社会经济状况、国籍、外貌、种族、宗教、性取向或技术选择而区别对待。

## 受到鼓励的行为

- 对不同经验水平保持耐心，用事实和可复现证据讨论问题。
- 给出具体、可执行且聚焦于代码或设计的反馈。
- 尊重项目边界、来源平台规则、隐私和知识产权。
- 主动说明不确定性、利益冲突以及 AI 辅助生成内容的范围。
- 在意见不一致时关注共同目标，并愿意修正自己的判断。
- 对新贡献者提供上下文和清晰的下一步。

## 不可接受的行为

- 侮辱、歧视、骚扰、威胁、跟踪或带有性意味的言行。
- 人身攻击、恶意揣测、持续打断或公开羞辱他人。
- 未经许可公开他人的个人信息、凭据或私密沟通。
- 诱导绕过访问控制、验证码、付费墙、平台政策或数据授权。
- 垃圾信息、恶意代码、供应链攻击或故意破坏项目协作。
- 在收到维护者停止要求后继续上述行为。

## 适用范围

本准则适用于仓库、Issue、Pull Request、代码评审、讨论区以及代表 HotKey 参与的公开或私下社区互动。

## 报告与处理

如遇不当行为，请通过维护者 GitHub 个人资料中列出的私密联系方式报告。不要在公开 Issue 中披露敏感个人信息。报告应尽量包含发生时间、位置、相关链接、行为描述和可用证据。

维护者会保护报告者隐私，并根据影响采取提醒、警告、临时限制或永久移除等措施。恶意或明显虚假的报告同样违反本准则。

## 执行原则

处理决定会考虑行为的影响、是否重复发生、是否愿意修正以及社区安全需要。维护者对其负责的社区空间拥有最终解释和执行权。

本准则参考 [Contributor Covenant 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct.html) 制定。
